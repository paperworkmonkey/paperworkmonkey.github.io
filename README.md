Tools for a DGH intensivist that may be helpful for others. One webpage functions as a feed calculator, the idea conceived as a guide for the initiation and a rough check for NG feeds on an adult ICU. The second page is an aide to finding ICNARC diagnosis codes using a search process rather than the tree.

Written in HTML, css and JS. Not pretty, could do with refactoring a lot, and as it all happens in the browser, a little server-side action too.
